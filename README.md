# roc-job-radar

Monitor Rochester-area employers for newly posted tech jobs.

## Overview

Roc Job Radar aggregates job postings from selected Rochester employers' career sites and highlights newly posted roles that may be relevant for software engineers.

The primary goal is to identify new job listings quickly so they can be reviewed and applied to early.

## Planned Features

- Scrape career sites from selected Rochester employers
- Detect newly posted jobs
- Filter and rank roles relevant to tech positions
- Simple interface for reviewing new listings
- Scheduled refresh of job data
